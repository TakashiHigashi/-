# データ

このディレクトリには研究で使用・作成したデータを格納します。

## ⚠️ 注意事項

- **大容量ファイル**: Git LFSを使用してください
- **機密データ**: リポジトリにコミットしないでください（`.gitignore`に追加）
- **データの出典**: 各データセットのメタデータを記録してください

## データ処理

データの前処理スクリプトは`preprocessing/`ディレクトリに格納してください。

```bash
# 前処理の実行例
python preprocessing/preprocess.py --input raw_data/ --output processed_data/
```

## メタデータ

各データセットには`metadata.json`を添付してください：

```json
{
  "name": "データセット名",
  "description": "説明",
  "format": "csv",
  "size": "100MB",
  "created_at": "2024-01-01",
  "source": "データの出典",
  "license": "ライセンス情報"
}
```
