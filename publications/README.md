## 📊 発表資料作成ガイドライン

### ディレクトリ構成

```
publications/
├── papers/              # 論文
│   ├── conference/      # 国際会議論文
│   ├── domestic/        # 国内学会論文
│   └── journal/         # ジャーナル論文
├── slides/              # プレゼンテーション資料
│   ├── oral/            # 口頭発表
│   └── invited/         # 招待講演
├── posters/             # ポスター
└── supplementary/       # 補足資料
```

### ファイル命名規則

- 論文: `YYYY_学会略称_第一著者名.pdf`
- スライド: `YYYYMMDD_発表場所_タイトル.pdf`
- ポスター: `YYYYMMDD_イベント名_タイトル.pdf`

## 📝 BibTeX

論文の引用情報は`bibtex/`ディレクトリに格納してください。
